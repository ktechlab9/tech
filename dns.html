DNS Explained (Beginner to Pro)

What is DNS?

DNS (Domain Name System) is like the phonebook of the internet. It converts domain names into IP addresses.

Example: google.com → 142.250.x.x

Why DNS is Needed

Humans remember names, computers use IP addresses. DNS connects both worlds.

How DNS Works (Step-by-Step)

  • You type a website (e.g. google.com)
  • Your device asks DNS server
  • DNS finds the IP address
  • Your browser connects to server
  • Website loads

Types of DNS Servers

  • Recursive Resolver (e.g. your ISP or public DNS)
  • Root Server
  • TLD Server (.com, .org)
  • Authoritative Server

Common DNS Servers

  • Google DNS: 8.8.8.8
  • Cloudflare DNS: 1.1.1.1
Note: 8.8.8.8 is NOT Google website — it is a DNS server.

Try It Yourself

Open terminal and run:

nslookup google.com
